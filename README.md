# MFC를 이용한 악기 연주 프로그램

### 주요기능
1. 원하는 악기 연주 (콤보박스, 지정된 키)
   - 피아노, 클라리넷, 첼로, 트럼펫, 하모닉기타, 일렉기타 
   
2. 옥타브 조절 (방향키, UI)

3. 악보 쓰기
   - 박자 조절 (마디 시간단위 조절)
   - ![image02](https://user-images.githubusercontent.com/71927210/126893543-6f075e79-d123-4272-bbc9-be66a69390b3.png)

   - 작성한 악보 재생
   - 작성한 악보 삭제
   - 작성된 악보 공유하기 (네트워크)
